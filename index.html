TryJS – Free Online JavaScript & TypeScript Playground

TryJS is a free JavaScript and TypeScript playground that runs in your browser. Write code and see results instantly, with no signup or installation. Use it to learn JS/TS, test ideas quickly, and share runnable examples.

What you can do

Switch between JavaScript and TypeScript with one click; TypeScript is compiled in-browser. The editor auto-runs as you type, and you can run instantly with Cmd+Enter. The console shows logs, warnings, errors, timing output, and REPL-style expression results.

Import npm packages directly, browse ready-to-run snippets, and share code as a URL or embeddable iframe. You can also export your code as an image with multiple frames, themes, paddings, and gradient backgrounds.

Choose from multiple syntax themes and editor fonts. Your code is saved locally per language, and execution runs in a sandbox with timeouts to prevent runaway scripts.

Why use this playground

This online JS/TS playground is built for fast experiments, learning, and sharing. There is no backend for your code execution: everything runs locally in your browser. It works on both desktop and mobile with a streamlined interface.

TryJS is open source. You can find the project on GitHub, browse a concise features page, and use the live app at tryjs.app.
